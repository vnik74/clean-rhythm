# Clean Rhythm // Аналитический дашборд менеджера

Внутренний интерфейс (панель управления) для мониторинга входящих заявок, анализа анкет клиентов и контроля эффективности автоматической воронки продаж премиального бренда ухода за кожей **Clean Rhythm**.

## 📊 Основной функционал
* **Мониторинг ключевых метрик:** Отслеживание посещаемости, конверсии сайта в анкеты, скачиваний лид-магнитов и прямых продаж наборов ухода.
* **Аналитика проблем кожи:** Наглядная диаграмма распределения основных запросов пользователей (сухость, чувствительность, тусклый цвет) для калибровки складских запасов и объемов сырья.
* **Лента лидов (CRM):** Интерактивная таблица последних поступивших анкет с отображением текущего статуса клиента в автоворонке сообщений (Telegram/WhatsApp).

## 🎨 Дизайн-система интерфейса
Дашборд спроектирован в фирменном стиле эко-минимализма бренда Clean Rhythm:
* **Background:** Светлый нейтральный фон (`#f7f6f4`) для снижения нагрузки на глаза менеджера.
* **Cards:** Белоснежные минималистичные карточки метрик (`#ffffff`) с обилием "воздуха" и whitespace.
* **Accents:** Благородный шалфеевый/оливковый цвет (`#4a5d4e`) для индикаторов прогресса, статусов и активных элементов.
* **Адаптивность:** Сетка автоматически перестраивается из 4-колоночной (десктоп) в 1-колоночную (мобильные устройства) без потери читаемости данных.

## 📂 Структура файлов репозитория
* `dashboard.html` — Главная страница интерфейса аналитической панели.
* `css/` — Стили оформления, кастомные прогресс-бары и медиа-запросы адаптивности.
* `images/` — Локальные графические элементы интерфейса (иконки, логотип бренда).

## 🛠️ Технический стек
* HTML5 (Семантическая разметка таблиц и метрик)
* CSS3 (Flexbox, Grid Layout, Fluid Media Queries)
* Среда разработки: **OpenCode**
